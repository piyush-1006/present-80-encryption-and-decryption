# present-80-encryption-and-decryption
I had implemented present 80 encryption(this is done to secure the data from being illegally seen from outsider) which transform the data given into another ciphertext using masterkey. I had written it in  C++ code and use it on ESP 32 S3 board and generated the QR code which stores the ciphertext and  mobile application that can retrieve data.
# key feature
writen the present 80 algorithm to encrypt the give data.
written the reverse algorithm to decrypt the data.
using ESP32 S3 i get the encrypted QR code.
made an app which can extract the data from QR code and decrypt it.
The app also take the masterkey to decrypt.
